Aprendendo Repetições em JavaScript

Esta página demonstra diferentes formas de criar repetições em JavaScript para contar de 1 até 10, com destaque especial para números pares usando <mark> e <b>.

1. Contagem com While

O loop while executa um bloco de código enquanto uma condição for verdadeira.

let i = 1;
while (i <= 10) {
  document.write(i + " ");
  i++;
}

2. Contagem com Do-While

O loop do-while é similar ao while, mas garante que o bloco de código seja executado pelo menos uma vez.

let j = 1;
do {
  document.write(j + " ");
  j++;
} while (j <= 10);

3. Contagem com For

O loop for é mais compacto, combinando inicialização, condição e incremento em uma única linha.

for (let k = 1; k <= 10; k++) {
  document.write(k + " ");
}

Bônus: Somente Números Pares

Demonstração de como filtrar e destacar apenas os números pares:

for (let n = 1; n <= 10; n++) {
  if (n % 2 === 0) {
    document.write("<mark><b>" + n + "</b></mark> ");
  }
}

Como identificar números pares?

Usamos o operador módulo (%) que retorna o resto da divisão. Se o resto da divisão por 2 for 0, o número é par:

if (numero % 2 === 0) {
  // É par
}
